# HTest

1 .- install packages:
```console
  npm install
```
2 .- run server:
```console
  npm start
```
3 .- in this case i decide to use the port `5001`
  so in the browser `http://localhost:5001`
  
4 .- For the test i use `vitest`, you can run the test with:
```console
  npm run test
```

  ## And that's it! have a nice day!
